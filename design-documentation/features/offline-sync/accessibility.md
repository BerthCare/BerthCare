# Offline Sync: Accessibility

## Screen Reader

Sync status announcements:

- "Saved locally" (when offline)
- "Syncing" (when syncing)
- "Synced" (when complete)
- "Sync failed. Tap to retry." (when error)

---

## Visual Indicators

Sync status uses color + icon + text (redundant encoding for colorblind users).
