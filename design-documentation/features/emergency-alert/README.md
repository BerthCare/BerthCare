# Emergency Alert Feature

## Overview

The emergency alert lets Sarah call for help instantly. In a real emergency, every second counts.

**Purpose:** Get Sarah on the phone with her coordinator in <2 seconds.

**Success criteria:**
- Emergency button always accessible
- Tap → Tap → Phone dials (2 taps, <2 seconds)
- No forms before the call
- Alert logged automatically

## Key Principles

1. **Call-first, data-second** – Phone call happens immediately
2. **Always accessible** – Emergency button on every screen
3. **No forms** – No dropdowns, no severity selectors
4. **Optional note** – Add context after call (not before)
5. **Automatic logging** – Alert logged in background

## Components Used

- Emergency button (`emergency-trigger.md`)
- Emergency modal (`modals.md`)
- Call button (large, red, unmistakable)
- Optional note input (after call)
