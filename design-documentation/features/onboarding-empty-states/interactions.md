# Onboarding: Interactions

## Contextual Hints

**Trigger:** First time user sees a screen

Behavior:

1. Hint appears (fade in, 300ms)
2. Stays visible for 3 seconds
3. Fades out (300ms)
4. Never shown again (stored in local preferences)

---

## Empty State Actions

**No visits today:** No action required (just informational).

**First visit:** Placeholders guide input format.
