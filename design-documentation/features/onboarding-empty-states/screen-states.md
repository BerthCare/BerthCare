# Onboarding: Screen States

## First Launch (Login)

Email + invitation code. No password (invitation-based auth).

---

## First Visit (Empty Fields)

All fields empty with helpful placeholders. Contextual hint: "Fields auto-save as you type."

---

## No Visits Today (Empty State)

Calendar icon + "No visits today. Enjoy your day off!"

---

## Contextual Hints

Brief tooltips that appear once, fade after 3 seconds. Never shown again.

**Examples:**
- "Tap a client to start documenting"
- "Fields auto-save as you type"
- "Swipe up to complete visit"
